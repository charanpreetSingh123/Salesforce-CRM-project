🚀 Salesforce CRM Automation System

📌 Overview

This project is a custom Salesforce-based CRM system designed to manage customer and lead data while automating key business processes.

It demonstrates a complete end-to-end implementation combining data modeling, automation, Apex programming, and analytics to simulate a real-world sales workflow.

⸻

🎯 Features

🔹 Custom Objects & Relationships
	•	Created My Customer and My Lead objects
	•	Implemented Lookup Relationship (Lead → Customer)
	•	Structured data for efficient CRM operations

⸻

🔹 Automation (Flow + Apex)
	•	Built a Record-Triggered Flow for lead qualification
	•	Developed an Apex Trigger to enforce business logic
	•	Automatically updates Lead Status → Qualified when a customer is assigned

⸻

🔹 Apex Logic
	•	Bulk-safe trigger handling multiple records
	•	Uses trigger context (before insert, before update)
	•	Executes only when required (optimized logic)

⸻

🔹 Reporting
	•	Created Qualified Leads Report
	•	Applied filters and grouping for meaningful insights

⸻

🔹 Dashboard
	•	Built an interactive dashboard with charts
	•	Displays key metrics for sales tracking
	•	Provides visual insights into lead qualification

⸻

🛠 Tech Stack
	•	Salesforce CRM (Lightning Experience)
	•	Apex (Triggers)
	•	Flow Builder
	•	Reports & Dashboards
	•	Object Manager

⸻

⚙️ Business Logic

When a lead is linked to a customer:
→ The system automatically updates the lead status to Qualified

This is implemented using both:
	•	Declarative automation (Flow)
	•	Programmatic logic (Apex Trigger)
